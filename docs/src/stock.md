# Stock data

GeoMakie offers coastlines and an image of the Earth from [Natural Earth](https://www.naturalearthdata.com/) as pre-loaded datasets, which you can get using `GeoMakie.coastlines()` or `GeoMakie.earth()`.

```@setup data
using GeoMakie
```

```@example data
GeoMakie.coastlines()
```

```@example data
GeoMakie.earth()
```
